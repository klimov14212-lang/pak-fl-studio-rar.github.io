# FL Packs Hub

Платформа для обмена паками для FL Studio.

## Структура проекта
- `index.html` - главная страница
- `css/style.css` - стили
- `js/app.js` - логика приложения

## Как использовать
1. Открой `index.html` в браузере
2. Загружай свои паки через форму
3. Скачивай паки других пользователей

## Технологии
- HTML5
- CSS3
- JavaScript (ES6+)
- IndexedDB для хранения файлов
- LocalStorage для метаданных
